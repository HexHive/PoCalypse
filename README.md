# PoCalypse

Repository of PoC inputs for different fuzzer benchmarks.

The directory structure is the following: library/program/PoCs


# Contributing

If you want to contribute to this repository, feel free to add PoCs from your fuzzing campaigns this this repo as a pull request.


# License 

We use a CC BY-ND 4.0 license to ensure that all new PoCs are contributed back to the repo instead of forking and creating new repos. This allows us to keep PoCs in a single place and organize discussions around them.
